# Pokemon coding task

### Setup

1. Clone the repository
`git clone https://github.com/DavidDrob/coding-task.git`
2. Change directory
`cd coding-task`
3. Install dependencies
`npm install` or `yarn`
4. Start the server
`npm run dev`
5. Go to localhost:3001

### Build

1. Build the dist folder
`npm run build`
2. Preview the website
`npm run preview`
